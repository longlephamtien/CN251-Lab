This repository contains all lab materials and reports for Computer Networks (LAB) CO3094 at HCMUT (VNU-HCM).  

---

## Session 1 - Networking Basics and the Application Layer

### **Lab 1: Introduction to Networking Devices and Tools**

#### **Lab 1a - Introduction to Wireshark Packet Sniffer Tool**
This introductory lab familiarizes students with **Wireshark**, a network packet analyzer.  
You will perform basic packet captures and observe network traffic.  
Wireshark captures (“sniffs”) packets sent and received by your machine, displaying detailed protocol fields without altering the communication.  

#### **Lab 1b - Introduction to Networking Devices**
**Objectives:**
- Understand basic networking devices (routers, switches, hubs, etc.)  
- Learn their core functions and interconnections  
- Build a simple physical or simulated network topology  

#### **Lab 1c - Socket Programming with Java**
**Objectives:**
- Practice **socket programming** concepts  
- Build a simple **client-server chat application**  
- Implement **multithreading** for concurrent communication  


### **Lab 2: Getting Started with Cisco Packet Tracer**

**Overview:**  
Cisco Packet Tracer is a powerful visual network simulation tool developed by Cisco Systems.  
It allows users to design and configure virtual networks using routers, switches, end devices, and multiple types of connections via an intuitive GUI or command line.  

**Key Features:**  
- Drag-and-drop topology building  
- End and network device variety (servers, PCs, routers, hubs, etc.)  
- Protocol support and configuration flexibility  
- CLI and GUI configuration interfaces  

#### **Lab 2a - Create a Simple Network Using Packet Tracer**
Build a simple network topology in Packet Tracer using routers, switches, and end devices.  
Verify device connectivity and basic configuration using CLI commands.  

#### **Lab 2b - Packet Tracer Tutorial**
Follow guided exercises to understand interface configuration, IP addressing, and cable types.  
Learn to troubleshoot network connectivity issues through simulation.


### **Lab 3: Application Layer**

#### **Lab 3a - Wireshark Lab: HTTP**
Explore the **HTTP protocol** in action using Wireshark.  
Analyze HTTP GET/response exchanges, message formats, and persistent connections.  
Investigate how large documents and embedded objects are retrieved and understand the basics of **HTTP authentication** and its security limitations.

#### **Lab 3b - Wireshark Lab: DNS**
Study the **Domain Name System (DNS)**, which maps hostnames to IP addresses.  
Examine client-side DNS behavior, query/response structures, and DNS caching using Wireshark.  
Understand **DNS records** (A, NS) and recursive vs. iterative resolution mechanisms.
